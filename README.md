Real Estate Market Trends Dashboard

1. Project Title / Headline

    A dynamic Power BI dashboard designed to analyze real estate market trends and identify profitable investment opportunities.

---

2. Short Description / Purpose

    This dashboard provides a comprehensive analysis of real estate data, helping users understand pricing patterns, investment potential, and market trends across     different locations. It enables data-driven decision-making for buyers, investors, and analysts.

---

3. Tech Stack

    - Power BI Desktop – Data visualization and dashboard creation
    - Power Query – Data cleaning and transformation
    - DAX (Data Analysis Expressions) – Calculations and measures
    - Data Modeling – Relationships between datasets
    - File Formats – .pbix (dashboard), .png (preview)
    
    ---

4. Data Source

    The dataset includes real estate property details such as:
    
    - Price
    - Location (Zip Code, Region)
    - Bedrooms and Bathrooms
    - Square Footage (Sqft)
    - Property Condition
    - Waterfront Availability
    
    This data is used to analyze pricing trends and investment opportunities.

---

5. Features / Highlights

    Business Problem
    Real estate investors often struggle to identify high-value properties and profitable locations due to lack of clear insights.
    
    Goal of the Dashboard
    To provide a clear and interactive visualization of real estate trends to support better investment decisions.
    
    Key Visuals
    
    - Map: Average Price by Location
    - Price vs Sqft Analysis
    - Top 5 Expensive Zip Codes
    - Best Value Zip Codes (Low Price per Sqft)
    - Price vs Property Age
    - Basement Impact on Price
    - 1 Floor vs 2 Floor Price Comparison

    Business Insights
    
    - Identified high-price and high-demand locations
    - Found best investment areas with low price per sqft
    - Observed impact of property features on price
    - Compared property types for better investment decisions
